About

I started learning to code in 2022. I did not have a clear direction at the time, but after finding Codecademy, I enrolled onto the learn Python 3 course. After spending some time with Python and becomming comfortable with the language I progressed onto the developing web apps with Django course. This was where I developed my interest in web development and decided to start the Full-stack Engineer course.

I have been working on Front-end development and have built skills using modern html, css and, drawing on my knowledge of Python, JavaScript. Using these skills, I am can build interactive, responsive and accessible web pages.

Having built a solid foundation, I will be learning how to build web applications using the React framework. I will then develop Back-end skills using Express.js and creating RESTful APIs.
